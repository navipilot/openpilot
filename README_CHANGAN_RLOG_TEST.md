# Changan RLog 测试脚本使用说明

## 简介

`test_changan_with_rlog.py` 是一个用于测试和验证 `opendbc/car/changan` 长安汽车代码实现的 Python 脚本。该脚本使用真实的 rlog（路由日志）驾驶数据进行验证。

## 功能特性

- ✅ 加载和解析 rlog 文件（支持 .rlog, .rlog.zst, .rlog.bz2 格式）
- ✅ 分析车辆参数（carParams）
- ✅ 测试 CAN 消息解析（CarState）
- ✅ 验证信号范围和有效性
- ✅ 测试控制消息生成（CarController）
- ✅ 生成详细的测试报告

## 使用方法

### 基本使用

1. 确保你在 openpilot 项目根目录下：
```bash
cd /path/to/openpilot
```

2. 运行脚本：
```bash
python test_changan_with_rlog.py
```

3. 按提示输入 rlog 目录路径：
```
请输入 rlog 目录路径 (例如: /home/jeff/realdata1/00000002--b52be56519--102):
> /home/jeff/realdata1/00000002--b52be56519--102
```

### RLog 文件格式

脚本支持以下 rlog 文件格式：
- `.rlog` - 未压缩的 rlog 文件
- `.rlog.zst` - Zstandard 压缩的 rlog 文件
- `.rlog.bz2` - Bzip2 压缩的 rlog 文件

### 输入路径示例

可以输入以下任一类型的路径：

1. **目录路径**（推荐）：
```
/home/jeff/realdata1/00000002--b52be56519--102
```
脚本会自动在目录中查找 rlog 文件

2. **直接文件路径**：
```
/home/jeff/realdata1/00000002--b52be56519--102/rlog.zst
```

## 测试内容

### 1. RLog 数据加载
- 验证路径是否存在
- 查找并加载 rlog 文件
- 统计消息类型分布

### 2. 车辆参数分析
从 rlog 中提取并显示：
- 车辆名称和指纹
- 质量 (mass)
- 轴距 (wheelbase)
- 转向比 (steerRatio)
- 中心到前轴距离 (centerToFront)

### 3. CAN 消息解析测试
- 提取 CAN 消息
- 统计 CAN 地址分布
- 按时间分组消息
- 测试解析功能

### 4. 信号验证
验证以下信号的范围和有效性：
- 车速信号 (0-250 km/h)
- 转向角 (-180° 到 +180°)
- 档位信号
- 踏板位置 (0-100%)
- CAN 消息计数器连续性

### 5. 控制消息生成测试
- 提取 sendcan 控制消息
- 统计控制 CAN 地址
- 验证控制输出

## 依赖项

脚本依赖以下模块（已包含在 openpilot 中）：
- `tools.lib.logreader` - RLog 文件读取
- `cereal.log` - 消息格式定义
- `opendbc.can.parser` - CAN 消息解析
- `opendbc.car.changan` - 长安汽车实现

## 故障排除

### 问题：找不到 rlog 文件
**解决方案**：
- 确认路径正确
- 检查目录中是否包含 .rlog, .rlog.zst 或 .rlog.bz2 文件
- 尝试使用完整的文件路径而非目录路径

### 问题：导入错误
**解决方案**：
- 确保在 openpilot 项目根目录下运行
- 检查 Python 环境是否正确配置
- 确认所有依赖项已安装

### 问题：未找到 CAN 消息
**解决方案**：
- 确认 rlog 文件包含实际的驾驶数据
- 检查 rlog 文件是否损坏
- 尝试使用不同的 rlog 文件
